QuiverAI

AI设计工具

AI矢量图形生成工具，输出可编辑的 SVG 代码

QuiverAI是什么 QuiverAI 是专注矢量图形生成的AI设计工具 ，基于自研基础模型 Arrow 构建的”视觉代码生成”平台，能直接输出可编辑的 SVG 代码，非传统像素图像。QuiverAI 覆盖 Logo、插画、字体和动画四大场景，面向设计师与开发者提供从文本或参考图生成生产级矢量资产的能力，支持通过 API 和 SDK 集成到第三方设计工具中。 QuiverAI的主要功能 文本转 SVG（Text-to-SVG）：输入自然语言描述即可生成 Logo、图标、插画等矢量图形，支持风格化指令（如扁平单色、圆角、描边风格）。 图像转 SVG / 智能矢量化（Image-to-SVG）：将 PNG、JPEG、WebP 等栅格图像转换为干净、结构化的 SVG 代码，支持自动裁剪主体（auto_crop）。 实时流式渲染：通过 Server-Sent Events 实时查看生成过程中的推理、草稿和最终成品。 多格式导出：支持导出 SVG、PNG、JPEG、WebP 四种格式。 自然语言迭代编辑：生成后可通过文字指令对 SVG 进行精细化修改，无需手动调整路径。 参考图风格约束：支持上传最多 4 张参考图像，引导输出风格与构图。 高保真模式（Arrow 1.1 Max）：针对复杂插画、技术图纸等细节敏感场景提供增强版模型。 如何使用QuiverAI 注册账号：访问 QuiverAI 官网 https://quiver.ai/ ，完成注册并登录。 选择生成模式：在创作界面选择”Text-to-SVG”（文生矢量）或”Image-to-SVG”（图生矢量）。 输入提示或上传素材： 文生模式：输入描述性提示词，可附加风格指令。 图生模式：上传 PNG/JPEG/WebP 参考图，可选开启 auto_crop。 配置参数：选择模型版本（Arrow 1.1 或 Arrow 1.1 Max）、设置导出格式、上传风格参考图（最多4张）。 生成与预览：点击生成，通过流式渲染实时查看输出过程。 迭代优化：使用自然语言指令对颜色、形状、布局进行微调。 导出或集成：下载 SVG/PNG 文件，或通过 API Key 接入自有系统。 QuiverAI的关键信息和使用要求 官网：https://quiver.ai 应用入口：https://app.quiver.ai/creations 文档中心：https://docs.quiver.ai GitHub：https://github.com/quiverai（提供 Node SDK 与 OpenAPI 规范） API 接入：Base URL 为 https://api.quiver.ai/v1，认证方式为 Authorization: Bearer 核心端点：POST /v1/svgs/generations（文生图）、POST /v1/svgs/vectorizations（图生矢量） 免费额度：Free 计划每周提供 20 张 SVG 生成额度 付费方案：Basic（$20/月，100张/周）、Pro（$40/月，250张/周）、Enterprise（定制） 使用要求：需具备基础英文提示词能力；API 调用需 Node.js 或 HTTP
