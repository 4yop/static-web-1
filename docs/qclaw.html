QClaw是什么 QClaw 是腾讯电脑管家团队基于 OpenClaw 打造的本地 AI 助手，支持 Mac 和 Windows 双端。QClaw 支持微信直联，扫码绑定后，能随时随地通过微信发指令，远程操控电脑完成任务。QClaw 内置 Kimi-2.5 等国产大模型，可自定义切换，背靠 5000+ Skills 生态，支持文件处理、自动开发、社媒运营、学术整理等场景。QClaw目前为邀请制内测阶段，可通过官方问卷申请邀请码。 QClaw的主要功能 微信直联操控：扫码绑定微信后，随时随地通过手机发送指令，远程操控电脑完成任务，无需携带电脑也能处理工作。 智能文件操作：可远程打开、编辑、计算本地文件，例如直接语音让 AI 打开 Excel 表格并求和统计。 自动开发编程：支持一句话创建项目、编写代码并自动提交到 GitHub，全程零手动操作。 社媒自动运营：支持自动生成内容并在小红书等平台发布，自动互动回复，帮助账号涨粉。 学术文献整理：支持自动检索论文、筛选高引文献、生成综述并导出 PDF 或 LaTeX 格式。 智能日程提醒：定时推送天气、日程等信息，并根据情况给出穿衣出行建议。 持续记忆学习：记住用户偏好和使用上下文，用得越久越懂用户的习惯和需求。 如何使用QClaw 下载安装：访问 QClaw 官网 https://claw.guanjia.qq.com/ 下载客户端，支持 Mac 和 Windows 双平台，目前平台处于内测阶段，需申请获取内测资格。 扫码绑定微信：打开软件后，用微信扫描屏幕上的二维码，完成关联绑定，此时微信中会多出一个”龙虾”好友。 发送指令开干：绑定成功后，可以在电脑上直接操作，或随时随地打开微信，向”龙虾”好友发送文字或语音指令，QClaw 会立即在本地电脑上执行相应任务，将结果返回微信。 安全建议：首次使用需授予系统级操作权限，建议设置强密码和身份验证 Token，限制仅可信联系人可发送指令，敏感操作开启二次确认，保障本地数据安全。 QClaw的核心优势 零门槛微信直联：无需复杂配置，扫码可绑定微信，打破传统 AI 助手必须在电脑前操作的限制，真正实现随时随地远程操控。 本地化部署保障隐私—：所有数据存储在本地电脑，不上传云端，从根本上解决用户对数据安全和隐私泄露的担忧。 开箱即用的 Skills 生态：内置 5000 多种技能，覆盖办公、开发、学术、生活等场景，无需自己调教，随取随用。 持续进化的专属记忆：具备长期记忆能力，会记住用户偏好和使用习惯，越用越顺手，成长为最懂你的个性化 AI 助手。 国产模型原生支持：内置 Kimi、DeepSeek、GLM 等国产优质大模型，同时支持自定义切换其他模型，灵活满足不同需求。 QClaw的应用场景 移动办公场景：人在外勤或通勤途中，微信一句话能远程打开电脑里的报表文件，自动计算汇总并返回结果。 社媒运营场景：自媒体从业者无需时刻盯着手机，让 QClaw 自动生成小红书笔记、一键发布，自动回复评论互动。 编程开发场景：开发者只需描述需求，QClaw 能自动创建项目框架、编写代码、提交到 GitHub 仓库，大幅缩短从创意到代码的实现路径。 学术研究场景：研究生或科研人员面对文献综述任务时，QClaw 可自动检索近年限定主题的论文，筛选高引文献，整理成格式规范的综述文档并导出，提升科研效率。 日常生活场景：支持设置每日定时提醒，自动推送天气状况并给出穿衣建议，成为比闹钟更贴心的个人生活助理。
