腾讯Marvis是什么 腾讯Marvis是腾讯应用宝团队推出的操作系统级 AI 助手，定位为”更懂你的 AI 管家”。腾讯Marvis跳出传统应用层，直接扎根系统底层，通过内置的六大 AI Agent（PM、文件管家、系统运维、应用专员、搜索专家、网页交互专家）实现任务自动拆解与并行执行。腾讯Marvis支持效率模式与隐私模式，可跨手机与电脑多端联动，让用户一句话完成系统设置、文件整理、应用操作等复杂任务。 腾讯Marvis的主要功能 多 Agent 协作办公：内置 PM（项目经理）、File Agent（文件管家）、Computer Agent（系统运维）、APP Agent（应用专员）、Search Agent（搜索专家）、Browser Agent（网页交互专家）六大角色，由 PM 自动拆解任务并分发给对应 Agent 并行执行。 系统级深度操控：可直接修改系统设置（如鼠标灵敏度、深色模式）、查询硬件配置、清理 C 盘、生成电池健康报告、排查网络故障等，操作直达操作系统底层。 文件智能管理：支持本地文件/图片内容搜索、图片内文字识别、AI 图库与 AI 文档库构建、文档格式转换、简历生成网页版、合同审查等。 跨端远程控制：手机可随时连接电脑，实时查看任务执行画面并接管操作；支持云手机协同，实现多端任务无缝衔接。 可视化办公室：提供”Marvis 办公室”界面，用趣味动画实时展示各 Agent 的工作状态。 双模式隐私保护：效率模式融合端侧响应与云端算力；隐私模式全部数据本地处理，文件零上传。 如何使用腾讯Marvis 下载安装：访问腾讯Marvis官网 https://marvis.qq.com/ ，根据操作系统下载客户端。 选择模式：根据需求选择”效率模式”或”隐私模式”。 下达指令：在对话框输入自然语言任务，PM Agent 会自动拆解并分发给对应 Agent。 查看进度：点击对话右侧”展开侧边栏”，查看工作日志、产出物与任务预览，或进入”办公室”查看各 Agent 实时工作状态。 跨端联动：在手机端登录同一账号，可远程查看电脑任务画面并下发新指令。 腾讯Marvis的核心优势 开箱即用，零配置门槛：无需像 OpenClaw 等工具那样手动搭建运行环境，下载安装可直接操作系统底层。 系统级权限，操作更深：不同于普通 AI 聊天工具，Marvis 可直接调用系统命令、修改注册表、扫描磁盘、生成电池报告。 趣味可视化交互：办公室场景化设计让多 Agent 协作过程可视化，降低用户对黑盒 AI 的不信任感。 隐私可控：双模式设计让用户在高效与安全之间自主选择，敏感文件可完全本地处理。 生态整合能力强：背靠腾讯应用宝团队，在应用识别、跨端协同、软件生态打通方面具备天然优势。 腾讯Marvis的使用示例 手机随时操控电脑 帮我更新电脑里的CS2，我回家要玩 一句话完成电脑设置 帮我去掉windows各种广告 腾讯Marvis的同类竞品对比 维度 腾讯 Marvis OpenClaw（开源框架） QClaw 定位 操作系统级个人 AI 助手 AI Agent 开源运行环境 基于 OpenClaw 的极简封装工具 使用门槛 开箱即用，无需配置 需自行搭建环境、写提示词 需安装配置，通过微信小程序交互 Agent 架构 内置 6 大专业 Agent，自动调度 依赖用户自定义 Agent 基于 OpenClaw 封装，功能相对单一 系统深度 系统级 MCP，可直接修改系统设置 取决于用户配置，深度不一 主要侧重远程控制与文件操作 可视化 办公室场景，实时查看 Agent 状态 无原生可视化界面 微信小程序内查看任务画面 隐私模式 支持纯本地模型，文件零上传 可本地部署，但需自行配置 数据上传云端处理 跨端能力 手机-PC 双向远程控制 需额外配置 手机远程控制 PC 腾讯Marvis的应用场景 系统运维：C 盘清理、电池健康检测、网络故障修复、硬件配置查询、系统设置修改。 办公提效：简历转网页版、文档格式转换、合同审查、运营数据分析、参考文献整理。 文件管理：AI 图库分类、图片内容搜索、本地知识库构建、重复文件清理。 跨端协作：外出时用手机远程更新家中电脑游戏、下载文件、启动应用。 信息监控：自动监控全网限免游戏、大厂情报、社会新闻并推送汇总。
