MiMo Code是什么 MiMo Code 是小米大模型团队开源的新一代 AI 编程助手，基于 OpenCode 采用 MIT 协议开源，内置限时免费的 MiMo-V2.5 模型。独创的持久记忆系统，通过独立 subagent 自动保存状态解决长会话”健忘”问题；Compose 模式支持一键完成设计、规划、编码、测试全流程；以及 Dream 命令实现 7 天周期记忆沉淀，让 AI 越用越懂你。支持语音输入和第三方模型接入，在 SWE-Bench 等真实编程测试中表现优于 Claude Code，且全面中文汉化，是国内开发者高性价比的 AI 编程新选择。 MiMo Code的主要功能 持久记忆系统：内置项目记忆、会话检查点、任务进度三重机制，由独立 subagent 自动保存状态，解决长会话”越用越忘”的难题，支持上百轮对话不丢关键信息。 Compose 模式：按 Tab 键切换，输入一个简单想法即可自动完成设计、规划、编码、测试、审查全流程，交付工业级成品代码。 Dream 记忆沉淀：内置 /dream 命令，每 7 天自动触发独立 Agent 整理历史会话，合并去重、验证路径、压缩记忆，让 AI 越用越懂你。 语音输入控制：基于 MiMo-V2.5-ASR 语音识别，支持口头输入 prompt、修改指令、发送执行等操作，全程无需碰键盘。 多模型兼容：内置 MiMo-V2.5 限时免费通道，同时支持接入 DeepSeek、Kimi、GLM 等主流模型及第三方 Token Plan。 全面中文汉化：所有设置项中文本地化，TUI 页面右侧常驻状态看板，随时观察工作进度，国内开发者友好。 开源免费：基于 OpenCode 采用 MIT 协议开源，个人可自由修改分发，企业可集成进自有工具链，无授权束缚。 智能代码编辑：支持代码库结构自动映射、安全变更分析、diff 审查与运行检查，在合并前捕获潜在问题。 如何使用MiMo Code 安装工具：Mac 和 Linux 用户执行 curl -fsSL https://mimo.xiaomi.com/install | bash 一键安装；Windows 用户通过 npm install -g @mimo-ai/cli 全局安装
