                CodeArts Doer是什么
CodeArts Doer 是华为推出的一站式AI编程助手，贯穿软件开发全生命周期。基于 6 大 Agent 智能体（团队、项目、产品、开发、测试、Committer 助手）实现全流程自动化协作，从需求管理到运维部署，大幅压缩开发时间，减少沟通成本和返工风险。CodeArts Doer融合华为多年软件研发经验，依托强大的研发数据积淀和深厚的技术基座，为开发者提供“更快、更准、更稳”的智能化研发提效方案。CodeArts Doer基础版提供智能编码助手，满足日常 IDE 编码效率提升需求；专业版在 CodeArts 平台内实现全流程智能辅助，提供从编码到部署的统一支持；企业版支持多智能体协作、需求/代码自主学习、思维链、时光机等高级能力，帮助开发者高效完成任务，提升开发体验。

CodeArts Doer的主要功能

全流程自动化协作：从需求管理到运维部署，基于 6 大 Agent 智能体实现全流程自动化，减少沟通成本和返工风险。

需求管理：通过团队助手Agent（Timmy）和产品助手Agent（Pony），实现需求的清晰规划、分解和详细分析，自动生成需求文档。
代码开发：开发助手Agent（David）支持内嵌于IDE，根据需求生成准确代码，提升开发效率。
测试管理：测试助手Agent（Cindy）自动生成测试用例，规划测试步骤和预期结果，确保交付质量。
运维部署：Committer助手Agent（Oliver）负责代码智能评审，确保代码符合规范，支持自动化部署。


多 Agent 协同：6 大 Agent 智能体互联互通，打破沟通障碍，提升信息传递的准确性和效率。
先进记忆算法与自学能力：具备记忆算法和自学能力，确保代码生成准确，适应项目需求变化。
全链路防范机制：内嵌代码防护和安全规则检查，保障代码安全性和可靠性，支持智能化测试与自动修复。
项目进度追踪与风险把控：提供项目进度报告和风险预警，帮助用户轻松掌控项目状态。
提升开发效率与质量：支持代码生成、测试用例生成和代码解释，减少手动编写代码的时间和错误，提高测试覆盖率和阅读代码的效率。

如何使用CodeArts Doer

注册与登录华为云：访问华为云官网，注册登录账号。用户也可以通过VS Code和JetBrains插件市场免费下载。
开通CodeArts Doer服务：在华为云控制台中找到CodeArts Doer服务并开通。
创建项目：在CodeArts Doer控制台中创建新项目，输入项目名称“Web应用开发”。
使用Agent智能体：

团队助手Agent（Timmy）：定义项目目标，分解任务为“前端开发”和“后端开发”。
产品助手Agent（Pony）：输入需求“开发一个用户登录页面”，生成需求文档。
开发助手Agent（David）：在IDE中输入“用户登录页面代码”，生成HTML和JavaScript代码。
测试助手Agent（Cindy）：输入测试需求“测试用户登录功能”，生成测试用例并执行测试。


Committer助手Agent（Oliver）：提交代码前，进行代码质量检查，确保代码符合规范。
项目管理与监控：使用项目助手Agent（Mike）跟踪项目进度，监控项目风险。
持续集成与部署：配置CI/CD流程，实现代码的自动构建、测试和部署。

CodeArts Doer的应用场景

快速开发：适用于需短时间上线的项目，如活动页面、小型工具等，通过多Agent协同，快速完成开发流程，节省时间和成本。
复杂项目管理：适合大型复杂项目，如多模块的企业级应用，借助Agent进行任务规划和进度追踪，确保项目按时交付。
敏捷开发：助力敏捷项目快速迭代，通过代码和测试用例生成加速迭代周期，提升团队响应速度。
代码质量提升：对代码质量要求高的项目，如金融、医疗系统，利用Agent生成高质量代码和测试用例，保障代码规范性和安全性。
跨部门协作：打破部门沟通障碍，实现信息高效传递和任务快速流转，提升协作效率。
