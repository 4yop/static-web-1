Hermes Agent是什么 Hermes Agent 是 Nous Research 推出的开源自主 AI Agent 框架，非简单的聊天机器人或编程助手，内置”自我改进学习循环”的智能体，能在完成任务后自动创建和优化可复用的 Skill，通过 FTS5 与 Honcho 实现跨会话的持久记忆，越用越懂用户。支持 15+ 消息平台统一接入、47 个内置工具、MCP 生态扩展，以及 Cron 定时自动化，可部署于本地、Docker、Modal 等 6 种终端后端。 Hermes Agent的主要功能 自我改进学习循环：完成任务后自动评估并生成可复用的 Skill 存入本地，使用中持续优化，下次遇到相似任务直接调用，官方称可降低约 90% API 调用成本。 持久三级记忆系统：基于 SQLite FTS5 实现毫秒级跨会话检索，配合 Honcho 进行用户偏好建模，记忆容量理论上无上限，越用越懂用户。 可移植技能系统：Skill 文档化存储于 ~/.hermes/skills/，兼容 agentskills.io 标准，支持社区共享与跨 Agent 复用。 15+ 消息平台网关：单一网关接入 Telegram、Discord、Slack、WhatsApp、Signal、钉钉、飞书、企业微信、Email、SMS、Home Assistant 等。 MCP 生态集成：支持 Model Context Protocol 服务器扩展，可安全接入 40+ 内置工具及外部 MCP 服务。 定时自动化：内置 Cron 系统，支持自然语言描述创建定时任务，结果可推送至任意已接入的消息平台。 多终端后端：支持 Local、Docker、SSH、Daytona、Singularity、Modal 等 6 种部署方式，Modal 支持无服务器休眠与按需唤醒。 200+ LLM 兼容：通过 OpenRouter 或直连 Anthropic、OpenAI、Ollama 等，支持故障自动切换，要求模型上下文不低于 64K tokens。 多 Agent 协作：支持 Subagent 并行隔离与委托，可将复杂任务拆分为子任务并协调多个 Agent 协同完成。 如何使用Hermes Agent 安装部署：在 Linux、macOS 或 Windows WSL2 上执行一行命令 curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash 即可完成安装，最低仅需 4GB 内存。 配置模型：运行 hermes setup 选择 LLM 提供商，支持通过 OpenRouter 接入 200+ 模型，或直连 Anthropic、OpenAI、Ollama 本地模型等，要求模型上下文窗口不低于 64K tokens。 启动对话：执行 hermes 启动 Agent，即可开始自然语言交互，Agent 会自动记录对话并建立用户模型。 接入消息平台：配置网关将 Hermes 接入 Telegram、Discord、Slack、钉钉、飞书、企业微信、Email 等 15+ 平台，实现跨渠道统一响应。 创建定时任务：使用内置 Cron 系统，通过自然语言描述设置定时任务（如”每天早上 9 点推送科技简报到 Telegram”），Hermes 会自主执行并推送结果。 扩展工具能力：通过 MCP 协议连接外部服务器扩展工具集，或直接使用 47 个内置工具完成网页搜索、浏览器自动化、代码执行等操作。 利用 Skill 复用：Agent 在执行复杂任务后会自动创建 Skill 并保存至 ~/.hermes/skills/，后续遇到相似任务直接调用；也可从社区导入或向 agentskills.io 贡献 Skill。 选择运行后端：根据场景选择 Local（本地）、Docker、SSH、Daytona、Singularity 或 Modal 后端部署，Modal 支持无服务器自动休眠与按需唤醒，适合低成本长期运行。 离线本地运行：配合 Ollama 加载本地大模型，可实现完全离线、零 API 费用的私有化部署，需 8–16GB+ VRAM。 Hermes Agent的官网地址 官网地址：https://hermes-agent.nousresearch.com/ Github仓库：https://github.com/NousResearch/hermes-agent Hermes Agent的核心优势 内置自我改进学习循环：完成任务后自动创建、评估并优化可复用 Skill，下次直接调用而非从零推理，官方称可降低约 90% 的 API 调用成本，实现”越用越聪明”。 持久三级记忆系统：基于 SQLite FTS5 毫秒级跨会话检索，配合 Honcho 辩证用户建模，记忆容量理论上无上限，能深度记住用户偏好与项目背景。 极简安装门槛：一行 curl 命令 60 秒内完成安装，5–15
