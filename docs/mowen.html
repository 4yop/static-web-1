墨问

AI写作工具

墨问是什么 墨问是专为创作者设计的AI笔记工具，提供便捷的写作与编辑功能。用户可通过官网或小程序进行笔记创作，支持多种文件类型上传，如图片、音频、PDF等。墨问集成了AI功能，如AI语音笔记和智能创作辅助，帮助创作者更高效地记录和整理信息。墨问Web版已全量开放，用户可直接在网页端进行创作，编辑器功能丰富，支持块选操作和多种样式设置。 墨问的主要功能 笔记创作与编辑：用户可以通过墨问官网或小程序进入写作状态，编辑器支持图片、画廊、底图、上传音频、黑体、引用、高亮和上传PDF等多种功能，提供四种不同的编辑器和阅读器样式供用户选择。 AI 功能集成：支持AI语音笔记、AI听读、AI润色等功能，帮助用户更高效地进行创作和内容优化。 多平台支持：除了Web端，墨问提供微信小程序版本，用户可以在不同设备上无缝切换，进行笔记创作和互动。 会员专属功能：会员可享受50G存储空间、无限制创建付费内容、生成小红书切图笔记、插入视频号等高级功能，非会员则有一定限制。 互动与分享：用户可以在小程序中对笔记进行点赞、评论等互动操作，Web端目前尚不支持互动功能，但可以通过小程序进行互动。 API 和MCP服务：为开发者提供API接口和MCP服务，支持与AI浏览器如Dia、豆包、Comet等的集成，方便用户在创作过程中使用AI能力。 如何使用墨问 进入写作状态：访问墨问官网 https://mowen.cn/ ，点击右上角的“写笔记”，或直接访问 https://w.mowen.cn/ 进入写作状态。 使用编辑器功能：编辑器上方工具栏包括图片、画廊、底图、上传音频、黑体、引用、高亮和上传PDF等功能。点击右上角的“A”可以设置编辑器和阅读器的样式，目前有四种样式可供选择，与小程序保持一致。 块选操作：写作过程中，鼠标滑至当前行左侧会出现块选图标，点击后出现下拉菜单，可以设置整段文字的样式。 保存与发布：写作过程中可以使用快捷键（Windows：Ctrl + S，Mac：Cmd + S）保存私密笔记并继续创作。点击右上角的“保存”按钮，保存文稿并进入阅读模式。私密笔记点击“发布”可以快速发布这篇笔记，状态为完全公开。在阅读状态下，双击可以进入编辑状态。 互动与分享：用户可以在阅读状态下，点击右上角的“小程序”按钮，进入小程序进行互动。 编辑已有笔记：如果想在Web端编辑已有的墨问笔记，可以点击小程序里墨问笔记的分享按钮，复制这篇笔记的Web链接到浏览器中进行编辑创作。 利用AI浏览器：在Web端写作可以充分利用AI浏览器的能力，如Dia、豆包、Comet等。这些工具可以帮助用户在写作过程中补充内容、润色、查错别字等。 墨问的应用场景 制作待办事项清单：清晰列出待办事项，方便管理和执行。 编写备忘录或笔记：用于记录各种信息，方便日后查阅。 保存重要信息或密码：安全地存储重要信息，避免遗忘。 记录会议或课堂笔记：在会议或课堂上快速记录要点，便于后续复习。 学术领域：教师可利用墨问在教学过程中通过提问激发学生思考，促进自主学习。 商务活动：通过墨问的方式向客户提问，更好地了解其需求，提供更精准的服务。
