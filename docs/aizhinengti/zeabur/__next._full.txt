                Zeabur是什么
Zeabur 是专为氛围编程设计的云部署AI 智能体。Zeabur 通过 AI 技术实现一键部署，支持多种编程语言和框架，包括前端、后端和数据库。用户通过简单的拖拽或代码推送快速上线项目。Zeabur 提供共享集群、自有服务器和托管 VPS 等多种部署选项，同时支持按量计费，帮助用户节省成本。Zeabur 具备自动扩展、对象存储、变量管理等强大功能，助力开发者专注于代码，无需操心复杂的部署细节。

Zeabur的主要功能

一键部署功能：用户通过简单的拖拽项目文件夹或浏览文件的方式，快速将代码部署到云端，无需复杂的配置。
AI 部署支持：与 Zeabur AI 对话即可完成部署，无需手动配置复杂环境，轻松实现项目上线。
多语言和框架支持：支持前端、后端、数据库等多种技术栈，包括大语言模型（LLM）生成的代码，满足不同开发需求。
多种部署选项：提供共享集群、自有服务器和托管 VPS 等多种部署方式，用户根据需求灵活选择。
零配置 CI/CD：代码推送到 GitHub 后，新版本将自动部署上线，无需手动配置持续集成和持续部署流程。
自动扩展功能：根据服务需求自动调整资源容量，确保应用在高流量时仍能稳定运行。
对象存储服务：提供类似于 S3 或 GCS 的对象存储服务，方便存储用户上传的文件，操作更简单。
变量和配置管理：在一个配置文件中管理所有复杂变量，无需多个 .env 文件，简化配置管理流程。
域名管理功能：用户能为部署的服务一键生成 Zeabur 提供的域名，或用自定义域名并添加 CNAME 记录。
文件管理功能：通过 Web 控制台的可视化界面轻松管理服务文件，操作体验类似云端硬盘。
模板市场：提供多种预设模板，用户能快速部署常见的服务，如 AI 聊天机器人、WordPress 等。
分享模板功能：用户能将自己的模板分享给他人，并通过模板市场从中获利。
按量计费模式：用户只需为实际使用的资源付费，无需购买整台服务器，节省成本。


如何使用Zeabur

创建账户：访问Zeabur官网，在首页点击“登录”按钮，点击“通过 GitHub 登录”，授权 Zeabur 访问 GitHub 账户。在 GitHub 授权页面点击“Authorize”按钮完成授权。
创建项目：授权成功后，自动跳转到 Zeabur 控制台，在控制台页面点击“创建项目”按钮。选择希望部署服务的区域（如 AWS 的不同地区）。系统自动生成一个随机项目名称，支持在项目设置中修改。
部署服务：点击刚刚创建的项目，进入项目详情页面。在项目详情页面点击“添加服务”按钮。
选择服务类型：

Git Service：部署源代码，Zeabur 自动构建和部署。
配置 GitHub 访问权限：如果是初次使用，需要为 Zeabur 配置 GitHub 访问权限。
Prebuilt Services：从常用服务市场中部署服务（如数据库或开源工具）。
查看服务市场：在 Zeabur 模板市场查看可用的服务列表。
Once Service：部署从 Once 购买的服务（如 Campfire）。


配置服务：在项目设置中管理环境变量，确保程序运行时能够正确读取配置。
绑定域名（可选）：在控制台中选择“域名管理”，绑定自定义域名或使用 Zeabur 自动生成的域名。
监控与优化：在控制台中查看服务的运行状态和日志。根据服务需求调整资源配置，如 CPU、内存等。

Zeabur的应用场景

初创企业快速上线：初创团队用 Zeabur 的一键部署功能，快速将产品上线，节省时间和精力，专注于产品迭代。
开发者个人项目：个人开发者通过 Zeabur 部署个人项目，无需复杂的服务器管理，轻松实现代码到云端的转变。
AI 项目部署：支持大语言模型（LLM）生成的代码部署，适合需要快速上线 AI 应用的开发者。
全栈开发：前端、后端和数据库等服务在 Zeabur 上一站式部署，适合全栈开发团队。
开源项目部署：通过 Prebuilt Services，用户能快速部署开源项目（如 WordPress、Supabase 等），适合开源爱好者。
