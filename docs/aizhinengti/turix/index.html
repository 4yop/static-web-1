TuriX

AI智能体

融合CUA（Computer Use Agent）的工业级通用 Agent 智能助手

TuriX是什么 TuriX 是融合CUA（Computer Use Agent）的工业级通用 Agent 智能助手，能跨应⽤程序和系统⾃动执⾏任务。用户通过自然语言可创建跨应用自动化工作流，无需编程或API。TuriX 支持定时任务执行，采用主动权限确认机制保障安全。相比海外同类产品，TuriX解决了国内无法使用、安全风险高、门槛高等痛点，在macOS和Windows上均可部署，端到端任务成功率达68%，且TuriX- CUA技术⽬前在GitHub上开源。 TuriX的主要功能 Chat 对话模式：支持用户通过自然语言进行智能问答、信息检索和日常交流，提供类聊天机器人的交互体验。 Work 办公模式：融合 CUA 与 CLI 技术，支持用户用自然语言描述自动生成跨应用自动化工作流并执行复杂办公任务。 Code 编程模式：提供代码生成、调试辅助和技术概念解释功能，帮助用户完成软件开发相关任务。 CUA 桌面操控：作为计算机使用代理直接控制电脑 GUI 界面，无需 API 可操作各类桌面应用程序和系统功能。 自然语言工作流：支持零编程经验的用户通过简单文字描述快速生成完整的自动化任务流程，无需编写代码或设计复杂逻辑。 定时任务调度：内置周期执行引擎，支持配置单次、每日、每周、每月或每年自动运行的定时自动化任务。 安全权限管控：采用审慎的授权机制，在所有涉及文件修改、网络访问等关键操作前主动向用户请求确认许可。 隔离运行环境：Parallel Computer 模式可在 Mac 上创建独立账户环境，使 AI 任务在隔离空间中运行而不影响用户主系统日常使用。 智能数据抓取：具备从网站和应用程序自动提取精准数据的能力，支持处理动态内容并绕过 CAPTCHA 验证。 跨平台兼容：支持部署在 macOS 15.6 及以上版本（需 Apple Silicon 芯片）和 Windows 10 及以上操作系统。 如何使用TuriX 下载安装：访问TuriX官网 https://turix.ai/ ，下载对应系统版本的安装包（macOS 为 .dmg 文件，Windows 为安装程序），按向导完成软件安装。 配置密钥：启动后在设置中填入 API Key 完成账户激活，无需配置其他环境参数或编写代码。 选择模式：在界面顶部标签栏切换 Chat（对话）、Work（办公）、Code（代码）或 CUA（桌面操控）四种工作模式。 输入指令：在对话框中用自然语言描述任务需求，如”整理桌面文件并发送给老板”或”查询今天的科技新闻”。 生成工作流：点击生成按钮，系统自动将自然语言拆解为可执行的操作步骤，展示完整任务流程。 设置调度：在 Work 模式下选择执行频率（单次/每日/每周/每月），设定具体运行时间和目标设备。 确认授权：当系统请求屏幕录制、文件访问或应用控制权限时，在弹窗中点击”允许”或”始终允许”继续执行。 监控任务：在左侧边栏查看任务执行进度，可随时手动中断正在运行的自动化流程。 开启隔离：Mac 用户建议启用 Parallel Computer 模式，让 TuriX 在独立账户环境中运行以保障主系统安全。 TuriX的关键信息和使用要求 产品定位：国内首款工业级通用 Agent + CUA（Computer Use Agent）融合产品，让 AI
