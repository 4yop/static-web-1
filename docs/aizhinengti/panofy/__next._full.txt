                Panofy是什么
Panofy 是开源项目 LLaMA-Factory 团队打造的全球首个可训练 AI 智能体平台，通过”训练”而非”提示”的方式，帮助企业和个人构建真正懂业务的专属智能体团队。与传统通用智能体不同，Panofy 支持用户基于自有知识库（PDF、Word 等）和工作流，一句话即可训练出具备持久记忆、并行执行和定时任务能力的 AI 助手，可组建具备不同技能的智能体团队协同工作，大幅降低 Token 消耗和幻觉问题。

Panofy的主要功能


智能体训练：基于知识库和工作流，一句话即可训练专属 AI 智能体，持续优化降低 Token 消耗。


并行任务执行：支持多个任务同时运行，无需排队等待，大幅提升效率。


定时任务调度：按设定时间自动触发任务，实现业务流程自动化，无需人工守候。


持久记忆系统：自动沉淀经验形成长期记忆，越用越懂业务，避免重复输入。


智能体团队：可训练多个具备不同风格与技能的智能体协同工作，减少幻觉问题。


知识库管理：支持 PDF、Word、TXT、Markdown 等多种格式文件上传，构建专属知识库。


团队协作：训练完成的智能体可通过链接分享给团队成员，受邀者直接使用。


多场景任务：支持 PPT 制作、代码编写、海报设计、研究报告、网页搭建、视频编辑等多种专业任务。


如何使用Panofy


访问官网：打开 Panofy 官网 https://panofy.ai/ ，支持 Google 或 GitHub 账号一键登录


邀请码激活：当前处于内测阶段，注册后需通过邮件获取邀请码完成激活。


创建智能体：用自然语言描述你想训练的智能体用途（如”帮我生成市场分析报告”）。


上传知识库：支持上传 PDF、Word、TXT、Markdown 等格式文件作为训练材料。


定义工作流：配置智能体的业务流程和操作规范。


启动训练：系统将基于你的知识库自动训练智能体。


持续优化：通过多次任务执行，智能体会自动沉淀经验，形成长期记忆。


效果评估：测试智能体输出质量，必要时补充知识库或调整训练参数。


即时任务：直接输入需求，智能体立即执行并返回结果。


并行任务：同时发起多个独立任务，利用并行执行能力提升效率（根据套餐支持 1-10 个并发）。


定时任务：设置特定时间自动触发任务，实现无人值守的自动化工作流。


创建多角色：针对 PPT 制作、代码编写、海报设计等不同场景，训练具备专业技能的独立智能体。


团队协作：让不同智能体协同处理复杂任务，减少单一智能体的幻觉问题。


生成分享链接：训练完成的智能体可生成专属链接。


邀请成员：将链接发送给团队成员，受邀者无需重新训练即可直接使用。


成本管理：分享后任务执行消耗的积分计入使用者账户，便于团队成本管控。

知识库维护：随时上传新文档或替换旧文件，保持知识库时效性。

存储管理：监控云端存储使用情况（FREE:10GB / PLUS:50GB / PRO:1TB），及时清理不再需要的文件。


Panofy的核心优势


可训练性优势：基于知识库和工作流训练专属智能体，大幅降低模型输出不确定性，一次训练持续生效。


持久记忆能力：自动沉淀任务经验形成长期记忆，无需重复输入背景信息，越用越懂业务。


高效并行执行：支持多个任务同时运行，无需排队等待，大幅提升任务处理效率。


智能体团队协作：可训练多个具备不同技能的智能体组成专家级团队，协同工作减少幻觉。


极简使用门槛：一句话描述即可创建智能体，无需复杂提示工程或代码配置，零门槛上手。


成本优化显著：通过训练实现高效上下文压缩，相比通用智能体大幅降低 Token 消耗。


企业级安全保障：提供云端安全环境，支持知识库集中管理，避免数据合规风险。


灵活团队协作：支持智能体链接分享，团队成员可直接使用，训练成果可标准化复用。


Panofy的关键信息和使用要求


开发团队：由知名开源项目 LLaMA-Factory 核心团队打造，技术背景深厚


产品定位：全球首个可训练智能体平台（Agent Training Platform），聚焦行业特定任务而非通用泛化


存储限制：FREE 用户 10GB、PLUS 用户 50GB、PRO 用户 1TB，超出后运行中的任务可能被中断


访问门槛：当前处于内测阶段，需通过官网注册后等待邮件邀请码激活账号


账号要求：仅支持 Google 或 GitHub 账号一键登录，暂不支持邮箱直接注册


知识库格式：支持上传 PDF、Word、TXT、Markdown 等主流文档格式构建训练知识库


并发限制：根据套餐等级限制并行任务数量（1-10 个），超出限制需等待当前任务完成

Panofy的应用场景


企业知识管理：上传 SOP、产品手册、培训文档，训练专属知识机器人，用于员工入职培训、客户支持问答、内部知识库查询。


内容营销创作：训练理解品牌调性的内容助手，自动生成符合品牌声音的营销文案、社媒帖子、广告创意和海报设计。

电商店铺运营：构建店铺客服智能体，自动处理售前咨询、订单查询、售后问题，支持定时执行店铺数据监控和营销任务。


软件开发辅助：训练代码助手理解项目规范和代码库，辅助编写代码、生成技术文档、进行 Code Review 和 Bug 修复。


网站页面搭建：根据业务需求快速生成响应式网页，包括落地页、产品展示页、表单页面等，支持从设计到代码的全流程。
