                ArkClaw是什么
ArkClaw 是字节跳动旗下火山引擎推出的云端托管版OpenClaw服务，主打”开箱即用、零配置、7×24小时在线”的轻量办公体验。用户无需本地部署、无需编写代码，打开网页即可直接使用，彻底解决原版OpenClaw环境配置繁琐、进程易中断等痛点。深度整合火山方舟模型平台，原生支持Doubao-Seed-2.0、Kimi 2.5等主流大模型，与飞书无缝集成，可直接在聊天框@智能体处理日程、多维表格及文档生成等任务。

ArkClaw的主要功能

云端托管：基于 OpenClaw 框架深度优化的 SaaS 版本，零本地部署、零代码编写、零环境配置，打开网页即可使用，支持 7×24 小时持续在线运行。
多模型协同：深度整合火山方舟模型平台，原生支持 Doubao-Seed-2.0 系列、Kimi 2.5、MiniMax 2.5 及 GLM 等主流大模型，可灵活切换不同模型处理各类任务。
飞书深度集成：支持飞书应用市场一键安装，可在飞书聊天框直接 @ArkClaw 处理日程预定、多维表格管理、复杂文档生成等办公任务，兼容 LUI 自然语言界面与 Terminal 双模式。
Skills 扩展能力：支持加载自定义 Skills 扩展智能体功能，火山引擎专属 Skills Hub 提供合规能力库，用户可按需增强智能体专业领域处理能力。
企业级安全：内置 Skills 安全扫描机制，采用云原生沙箱隔离技术，高危操作强制二次确认，直连 TOS 对象存储实现数据长效安全存储。
弹性算力规格：提供轻量版（2核4G）、标准版（4核8G）、高级版（8核16G）三档配置，适配个人基础使用到复杂多智能体并发任务的不同场景需求。

如何使用ArkClaw

网页端快速部署：登录火山引擎官网或方舟体验中心 arkclaw ，订阅 Coding Plan（Lite 或 Pro 版），点击”立即创建”按钮，约1-2分钟即可完成云端部署，无需任何本地配置。
飞书集成配置：在飞书开放平台创建企业自建应用，添加机器人能力，获取 App ID 和 App Secret，返回 ArkClaw 控制台填写对应信息完成配对授权。
飞书场景使用：在飞书聊天框中直接 @ArkClaw 智能体，通过自然语言指令处理日程预定、多维表格管理、复杂文档生成等办公任务，支持 LUI 对话模式与 Terminal 命令行模式双切换。
模型选择切换：在 ArkClaw 界面中选择火山方舟平台支持的模型（如 Doubao-Seed-2.0、Kimi 2.5、MiniMax 2.5、GLM 等），根据任务复杂度灵活切换以获得最佳效果。
Skills 扩展加载：访问火山引擎 Skills Hub 浏览合规能力库，或上传自定义 Skills 文件，按需扩展智能体的专业领域处理能力，如数据分析、代码生成、内容创作等。
任务执行与监控：输入任务指令后，ArkClaw 在云端沙箱环境中异步执行，高危操作会触发二次确认弹窗，任务结果自动同步至飞书或保存在 TOS 对象存储中供随时查看。
算力规格调整：根据任务需求在控制台切换轻量版、标准版、高级版三档配置，复杂多智能体并发任务建议选择高级版（8核16G）以获得更流畅体验。
