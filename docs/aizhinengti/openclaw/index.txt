                Moltbot是什么
Moltbot（原Clawdbot） 是开源免费的个人 AI 助手，能在个人电脑（如 Mac、Windows、Linux）上运行，支持通过 WhatsApp、Telegram、iMessage 等聊天软件接收指令并直接操作电脑完成任务。Moltbot能整理文件、修复代码、管理邮件，支持接入 Anthropic、OpenAI 等顶尖模型。Moltbot 具备永久记忆功能，能记住用户偏好和上下文自我进化，根据新指令开发新技能。Moltbot 强大的灵活性和扩展性使其成为提升个人效率和生产力的理想工具。

Moltbot的主要功能

任务执行：支持通过聊天软件（如 WhatsApp、Telegram、iMessage）接收指令，直接在本地电脑上执行任务，如整理文件、修复代码、发送邮件、管理日程等。
多平台支持：Moltbot 能运行在 Mac、Windows、Linux 等多种操作系统上，支持接入 Anthropic、OpenAI 等主流 AI 模型，以及本地模型。
持久化记忆：具备强大的记忆功能，能记住用户的偏好、上下文信息和任务细节，实现跨会话的持久记忆。
自我进化：用户可教授技能或操作逻辑，Moltbot 能内化为永久技能，在后续任务中应用。
系统操作：Moltbot 拥有与人类用户同等级别的系统权限，可直接调用终端、编写脚本、安装软件工具和控制智能家居设备。
网关集成：Moltbot 通过网关连接各类聊天平台，实现指令与反馈的实时传递，支持多智能体协作，能根据需求克隆出多个智能体。
技能扩展：支持丰富的技能插件，用户可根据需求扩展其功能，例如网页调研、浏览器自动化、文件管理等。
自动化工作流：支持设置定时任务和自动化工作流，例如定时检查邮件、自动执行代码测试、生成报告等。

如何使用Moltbot

检查前置条件：确保系统已安装 Node.js（版本 >=22）和 pnpm（可选，推荐用于从源代码构建）。
安装 CLI：通过运行 curl -fsSL https://molt.bot/install.sh | bash 或在 Windows 上运行 iwr -useb https://molt.bot/install.ps1 | iex 安装 Moltbot CLI。
运行引导向导：执行 Moltbot onboard --install-daemon，设置模型认证（推荐使用 OAuth）、网关配置、聊天通道（如 WhatsApp、Telegram）和配对默认值。
启动网关：如果在引导向导中安装了后台服务，网关应已自动运行；手动启动可用 moltdbot gateway --port 18789 --verbose。
快速验证：运行 moltbot status 和 moltbot health，检查网关状态和健康情况。
连接第一个聊天平台：用 moltbot channels login 登录 WhatsApp（扫描 QR 码），或手动配置 Telegram、Discord 等平台的令牌。
配对审批：首次通过 DM 发送消息时，需通过 moltbot pairing approve <platform> <code> 批准配对，确保消息被处理。
从源代码运行（开发用途）：克隆仓库 git clone https://github.com/moltbot/moltbot.git，安装依赖并构建项目，运行 moldbot onboard --install-daemon。
端到端验证：在新终端发送测试消息，如 moltbot message send --target <phone_number> --message "Hello"，确保 Moltbot 响应。
可选扩展：安装 macOS 菜单栏应用、iOS/Android 节点（支持 Canvas、相机、语音），或设置远程访问（如 SSH 隧道、Tailscale）。

Moltbot的应用场景


个人效率提升：通过聊天软件发送指令，Moltbot 可管理日程、整理文件、搜索信息，提升个人工作效率。


工作自动化：Moltbot 能辅助代码开发、处理邮件、生成项目报告，实现工作流程自动化。


家庭自动化：工具控制智能家居设备、管理家庭购物清单、自动下单，让家庭管理更加便捷。


学习与研究：用于整理学术资料、提供语言学习辅助、生成学习笔记，助力学习与研究。


创意与内容创作：用在辅助写作、设计、视频制作，提供创意灵感和内容优化建议。
