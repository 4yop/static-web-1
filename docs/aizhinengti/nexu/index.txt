                nexu是什么
nexu 是免费开源的 OpenClaw 桌面客户端，用户可通过图形界面将 AI Agent 接入微信、飞书、Slack 和 Discord。nexu 采用本地优先架构，数据 100% 本地保留，支持 Gemini、Claude、ChatGPT 等多模型一键切换，可自带 API Key 免登录使用。nexu 作为最快接入 Seedance 2.0 的开源龙虾，nexu无需命令行，双击即用，适合个人开发者和小团队打造”一人公司”的 AI 工作流。

nexu的主要功能

IM 无缝接入：支持一键连接微信、飞书、Slack、Discord，让 AI Agent 在手机端 24/7 随时待命。
模型自由切换：支持 Gemini、Claude、ChatGPT 等顶级模型一键切换，可 OAuth 或自带 API Key 免登录使用。
本地优先架构：所有数据 100% 存储在本地设备，业务信息不经过第三方服务器，代码开源可审计。
零门槛部署：纯图形界面无需命令行，双击安装即用，支持 macOS（含 Apple Silicon）和 Windows。
丰富 Skills 生态：原生集成 OpenClaw 和飞书 Skills，开箱即处理代码调试、内容创作、数据分析等复杂任务。
最快接入新模型：作为开源 OpenClaw 客户端，率先支持 Seedance 2.0 等最新模型能力，无需等待官方更新。

如何使用nexu

下载安装：访问 nexu.io 官网 https://nexu.io/ 或 GitHub Releases，下载对应系统版本（macOS Apple Silicon/Intel 或 Windows），双击安装，无需配置环境变量或终端操作。
初始配置

方式 A：注册 nexu 账号登录，Beta 期可免费使用内置的 Gemini 等顶级模型。
方式 B：点击”添加 API Key”，填入 Claude、ChatGPT、MiniMax 等自有密钥，无需注册可使用。


连接 IM 开始使用

点击”连接”选择平台：微信（需 8.0.7 版 OpenClaw 插件，扫码绑定）、飞书、Slack 或 Discord。
连接成功后，直接在手机 IM 里@Agent 或私聊，可远程操控电脑处理任务（写代码、查资料、生成内容等）。



nexu的关键信息和使用要求


定位：开源 OpenClaw 桌面客户端，让 AI Agent 接入微信/飞书/Slack/Discord。


核心优势：本地优先（数据不出设备）、MIT 开源可审计、支持 Seedance 2.0 等新模型快速接入。


成本：客户端永久免费，可 Beta 期免费使用内置模型，或自带 API Key 按量付费。


系统：macOS 12+（支持 Apple Silicon/Intel）或 Windows 10+，磁盘空间约 500MB。


微信连接：需微信 8.0.7 版本并开启 OpenClaw 插件，扫码即可绑定。


nexu的核心优势

完全开源可控：采用MIT 协议开源，代码可审计分叉，数据 100% 本地存储，拒绝厂商锁定。
最快接入新模型：率先支持 Seedance 2.0 等最新模型，内置 Gemini、Claude、GPT 等多模型一键切换，可自带 API Key 免登录使用。
原生 IM 集成：一键连接微信（扫码绑定）、飞书、Slack、Discord，Agent 24/7 在线，手机随时远程操控电脑。
零门槛体验：纯图形界面，无需命令行或配置环境，双击安装即用，支持 macOS/Windows 双平台。
