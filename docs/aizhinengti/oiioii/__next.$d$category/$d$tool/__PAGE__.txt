                OiiOii是什么
OiiOii 是全球首个专业动画创作Agent，能帮助创作者轻松实现动画梦想。OiiOii 通过内置的 7 个专业 Agent（如艺术总监、编剧、分镜师等），实现从剧本创作到动画生成的一站式服务，支持生成图片、视频、动画短片等多种形式。OiiOii 适合内容创作者和动画爱好者使用，目前OiiOii 处于内测阶段，用户可通过官网申请邀请码体验。

OiiOii的主要功能

一键生成动画：用户输入简单指令或提示词，OiiOii 能快速生成动画短片、视频、图片等内容，降低创作门槛，实现高效创作。
多角色智能协作：内置艺术总监、编剧、分镜师等 7 个专业 Agent，各司其职，协同完成动画创作，流程清晰且高效。
剧情故事短片：支持通过文字或上传角色参考图生成具有完整剧情的动画短片，涵盖角色设计、分镜生成和视频合成。
音乐概念短片：支持根据用户输入的音乐风格或主题，生成动态音乐视频（MV），支持节奏感和视觉效果的定制。
漫画转视频：支持将漫画内容转化为动画视频，实现静态漫画的动态化呈现。
衍生品设计：为角色设计周边产品，如抱枕、徽章等，满足创作者的衍生创作需求。
丰富风格库：提供近 200 种预设艺术风格，支持通过参考图或导演风格生成特定风格的动画，满足个性化需求。
批量生成与 IP 运营：支持基于同一 IP 的批量内容生成，保持角色和风格一致性，适合系列化内容生产。
互动反馈与实时修改：创作过程中用户能随时提出修改意见，Agent 根据反馈调整生成内容，确保满足需求。
多场景应用：适用剧情动画、音乐 MV、知识科普、IP 二创等多种场景，满足不同用户的多样化需求。
多平台支持：目前支持网页端使用，兼容电脑、手机和平板设备，方便随时随地创作。
两种生成模式：托管模式，支持用户全权委托自动生成内容；对话模式，支持用户与系统进行交互式调整，以精细化控制内容生成过程。


如何使用OiiOii

获取邀请码：访问OiiOii官网 https://www.oiioii.ai ，申请邀请码，或通过官方社群、社交媒体互动获取。
注册登录：用邀请码完成注册，登录 OiiOii 官网，推荐使用电脑端获得最佳体验。
输入创作需求：在首页输入文字指令或提示词，描述想要的动画内容（如风格、主题、时长等）。
选择创作模式：根据需求选择剧情短片、音乐 MV、知识科普、动态壁纸等创作模式。
与 Agent 协作：OiiOii 的智能 Agent（如艺术总监、编剧、分镜师等）会根据需求逐步生成内容，用户能实时查看进度，提供反馈。
调整与优化：在生成过程中，用户能随时修改指令或反馈意见，Agent 根据反馈调整生成内容。
预览与下载：生成完成后，用户能预览动画效果，满意后点击下载保存至本地。

OiiOii的应用场景


剧情动画短片：快速生成有趣的故事动画，适合自媒体创作者、动画爱好者制作短剧或微电影。


音乐 MV 制作：根据音乐风格和歌词生成匹配的动画 MV，为音乐创作者提供可视化内容。


知识科普动画：制作生动的科普视频，帮助教育工作者和知识博主将复杂内容用动画形式呈现。


IP 二创与衍生：基于经典 IP 或热门作品进行二次创作，适合自媒体创作者和粉丝群体快速生成创意内容。


动态壁纸与素材：生成个性化的动态壁纸或动画素材，满足设计师和创意工作者的需求。
