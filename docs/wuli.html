呜哩是什么 呜哩是阿里巴巴推出的AIGC创意生产力平台，基于通义千问团队的多个AI图像和视频模型，如Qwen Image 25.12模型进行高精度图像生成、Qwen Image Turbo实现极速图像生成，以及Qwen Image 25.11支持智能图像编辑，用户能通过简单的文本描述快速生成高质量的图片和视频。呜哩能帮助内容创作者、设计师以及普通用户，将创意想法转化为现实，简化设计流程并提高工作效率。呜哩的设计理念是将复杂的设计工作简化为易于操作的模板化路径，让用户能轻松地在多场景中应用生成的内容。 呜哩的主要功能 图片生成：用户输入文本描述后，呜哩AI用人工智能技术生成相应的高质量图片 视频生成：呜哩提供视频内容的生成服务，满足用户在不同媒体形式上的需求。 联想助手：根据用户输入的关键词提供相关的创意灵感和设计参考，帮助用户拓展创意边界。 翻译工具：内置的翻译工具可以将设计内容翻译成多种语言，以适应不同地区用户的需求。 资源库：包含海量模板、素材和风格参考，进一步降低创作门槛，用户可以直接使用这些资源快速完成设计工作。 如何使用呜哩 访问平台：访问呜哩官网 https://wuli.art/ ，注册和登录账号。 选择生成类型：在呜哩平台上，选择“图片生成”或“视频生成”功能。 输入创意描述：在输入框中输入创意描述，例如“香氛电商场景”或“电影风格海报”。平台将根据描述生成相应的图像或视频。 选择模型：从提供的模型中选择一个，例如Qwen Image 25.12、Qwen Image Turbo或Qwen Image 25.11，模型分别用于不同的生成和编辑需求。 利用辅助工具：如果需要，用联想助手来获取创意灵感，或使用翻译工具适配多语言内容。 生成内容：点击“生成”按钮，平台将根据描述和选择的模型生成内容。 编辑和优化：如果需要，用智能编辑功能对生成的图像进行进一步的优化和调整。 下载和使用：完成编辑后，下载生成的图片或视频，在你的项目中使用它们。 呜哩的使用示例 图片生成 三只精致牛角包叠放在木质托盘上，每只都拥有独特的装饰与口感。最下方的牛角包表面用白巧克力拉出网格纹路；中间一只撒有糖粉，并点缀着彩色糖果小花；最上方一只淋有融化的黑巧克力，撒上碎坚果，顶部还有一颗鲜红樱桃。所有牛角包金黄酥脆，层次分明，外皮油亮诱人。柔和自然光突出其细腻纹理与丰富细节，背景温暖柔和，虚化处理增强主体聚焦。整体画面高清细腻，8K分辨率，专业美食摄影风格，充满优雅与甜蜜氛围。 视频生成 一位白发考古学家在触摸古老陶片时，瞬间被拉入千年前制陶者的记忆幻象。画面从冷色调的考古现场，穿越到暖色调的古代生活，经历了制陶、爱情与战乱。电影感，情感深沉，光影对比强烈，细节丰富，4K，超现实主义。 呜哩的产品定价 呜哩现在为Beta测试阶段，免费向用户提供使用。 呜哩的应用场景 电商设计：为电商平台创建产品展示图、促销海报、广告横幅等，如“香氛电商场景”或“烘焙产品”。 社交媒体内容：平台能生成适合在社交媒体上分享的图片和视频，如“圣诞氛围”或“中式美学海报”。 营销和广告：平台能制作营销材料，包括宣传册、传单、广告牌等，例如“电影风格海报”。 个人创作：艺术家和设计师用呜哩探索新的风格和创意，如“水彩丽人插画”或“3D艺术字体”。 教育和培训：创建教育材料和演示文稿的视觉元素，吸引学生和参与者的注意力。
