Open Design是什么 Open Design 是开源本地优先的 AI 设计工作空间，定位为 Claude Design 的替代品。不捆绑 AI 模型，是直接调用你电脑上已有的编程 Agent，如 Claude Code、Cursor、Kimi、Codex 等 21+个，通过 SKILL.md 技能和 DESIGN.md 设计系统驱动工作流。可以在桌面端完成需求定义、视觉方向锁定、产物生成与评审的完整闭环，输出网页原型、仪表盘、演示文稿、图片及视频，导出为 HTML、PDF、PPTX 或 MP4。 Open Design的主要功能 开源本地优先：Apache-2.0 许可证，是 Claude Design 的开源替代品，所有数据保存在本地，无需依赖单一厂商云服务。 Agent 原生驱动：支持 21+ 种编程 Agent（Claude Code、Cursor、Kimi、Codex、Gemini、Copilot、DeepSeek 等），通过 MCP 协议一键接入。 技能系统：内置 259+ 个基于 SKILL.md 的可复用设计技能，拖入文件夹即可加载，Agent 可直接读取执行。 设计系统：提供 142+ 个品牌级 DESIGN.md 设计系统（Linear、Stripe、Apple、Vercel 等），支持上传截图或 Figma 文件自动提取品牌规范。 多形态产物输出：可生成网页原型、实时仪表盘、演示文稿（Decks）、图片、视频及 HyperFrames 动态图形。 多格式导出：产物支持导出为 HTML、PDF、PPTX、MP4 等工程可直接使用的格式。 BYOK 支持：自带 API Key，兼容 OpenAI、Anthropic、Google、Moonshot、OpenRouter、自托管 vLLM 等任意端点。 四步设计闭环：Detect（自动检测 Agent）→ Discover（30 秒需求问卷）→ Direct（锁定视觉方向）→ Deliver（流式生成与预览）。 如何使用Open Design 下载安装：访问 Open Design 官网 https://open-design.ai/ ，下载桌面应用（macOS / Windows），或源码克隆 github.com/nexu-io/open-design 用 Node 24 + pnpm 安装，也可 docker compose up 一键部署。 接入编程 Agent：安装后运行 od mcp install 将本地 Agent（Claude Code、Cursor、Kimi、Codex 等 21+）一键接入 Open Design 的 MCP 服务器。 启动守护进程：打开应用后守护进程自动扫描 $PATH 中的 Agent，加载 259+ 技能和 142+ 设计系统。 创建项目：在 Home 页选择产物类型（原型、仪表盘、演示文稿、图片、视频等），绑定一个 DESIGN.md 设计系统（如 Stripe、Apple 风格或自定义品牌）。 锁定需求：回答 30 秒问卷（用途、受众、调性、规模、品牌背景），快速对齐设计方向。 选择视觉方向：从 5 个确定性视觉方案中挑选，系统自动锁定 OKLch 色板、字体栈和布局姿态。 生成与预览：Agent 将产物写入本地磁盘，在沙盒 iframe 中实时预览真实 CSS 渲染效果。 评审迭代：通过内置的 5 维自评体系（哲学、层级、执行、特异性、克制）对产物进行批判性优化。 导出交付：确认后导出为 HTML、PDF、PPTX、MP4 或 ZIP，产物直接留在你的项目目录中。 Open Design的核心优势 完全开源免费：Apache-2.0 许可证，无订阅费，可自由 fork 和自托管。 本地优先，数据自主：所有产物和项目数据保存在本地文件系统，不经过厂商云端，隐私可控。 Agent 无关，自由切换：支持 21+ 种编程 Agent（Claude、Cursor、Kimi、Codex 等），技能和系统不绑定特定模型，换 Agent 只需改一行配置。 BYOK 成本控制：自带 API Key，直接向所选提供商付费，无中间商抽成，支持 OpenRouter 和自托管模型。 文件即设计系统：基于 SKILL.md 和 DESIGN.md 的声明式文件系统，技能和设计系统可复用、可版本管理、可在团队间共享。 真实代码渲染：输出的是真实 CSS、真实字体和真实组件，不是画布像素，产物可直接进入工程流程。 多形态一键产出：同一设计系统可流式生成网页原型、仪表盘、PPT、图片、视频及 HyperFrames。 品牌迁移零成本：上传截图或 Figma 导出文件，Agent 自动提取为 DESIGN.md，后续所有技能自动套用该品牌规范。 Open Design的同类竞品对比 对比维度 Open Design 快速产品原型设计：使用 259+ 技能和 142+ 设计系统，通过自然语言描述快速生成 Web、桌面或移动端的高保真交互原型，真实 CSS 渲染可直接用于开发交接。 品牌视觉系统构建与迁移：上传品牌截图或 Figma 导出文件，Agent 自动提取为 DESIGN.md，建立可复用的品牌规范，确保所有产物（原型、PPT、图片）视觉一致性。 演示文稿与提案制作：使用 Magazine Decks
