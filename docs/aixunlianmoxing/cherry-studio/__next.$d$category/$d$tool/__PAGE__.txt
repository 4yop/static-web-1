                Cherry Studio是什么
Cherry Studio 是集多模型对话、知识库管理、AI 绘画、翻译等功能于一体的开源全能 AI 客户端助手。Cherry Studio支持 Windows、macOS 和 Linux 系统，降低技术门槛，让零基础用户也能快速上手，同时为开发者提供强大的扩展能力。Cherry Studio 核心功能包括一问多答、自动分组、对话导出、AI 绘画、翻译功能等。Cherry Studio支持多服务商统一管理、高度自定义界面和布局，及本地知识库系统，确保数据安全。Cherry Studio 致力于提升用户的工作效率和创造力。

Cherry Studio的主要功能

基础对话功能：

一问多答：支持同一问题通过多个模型同时生成回复，方便用户对比不同模型的表现。
自动分组：每个助手的对话记录会自动分组管理，便于用户快速查找历史对话。
对话导出：支持将完整对话或部分对话导出为多种格式（如 Markdown、Word 等），方便储存与分享。
高度自定义参数：支持用户填写自定义参数，满足个性化需求。
助手市场：内置千余个行业专用助手，涵盖翻译、编程、写作等领域，同时支持用户自定义助手。
多种格式渲染：支持 Markdown 渲染、公式渲染、HTML 实时预览等功能，提升内容展示效果。


多种特色功能集成：

AI 绘画：提供专用绘画面板，支持用户用自然语言描述生成高质量图像。
AI 小程序：集成多种免费 Web 端 AI 工具，无需切换浏览器即可直接使用。
翻译功能：支持专用翻译面板、对话翻译、提示词翻译等多种翻译场景。
文件管理：对话、绘画和知识库中的文件统一分类管理，避免繁琐查找。
全局搜索：支持快速定位历史记录和知识库内容，提升工作效率。


多服务商统一管理机制：

服务商模型聚合：支持 OpenAI、Gemini、Anthropic、Azure 等主流服务商的模型统一调用。
模型自动获取：一键获取完整模型列表，无需手动配置。
多秘钥轮询：支持多个 API 秘钥轮换使用，避免速率限制问题。
精准头像匹配：为每个模型自动匹配专属头像，提升辨识度。
自定义服务商：支持符合 OpenAI、Gemini、Anthropic 等规范的三方服务商接入。


高度自定义界面和布局：

自定义 CSS：支持全局样式自定义，打造专属界面风格。
自定义对话布局：支持列表或气泡样式布局，能自定义消息样式（如代码片段样式）。
自定义头像：支持为软件和助手设置个性化头像。
自定义侧边栏菜单：用户根据需求隐藏或排序侧边栏功能，优化使用体验。


本地知识库系统：

多种格式支持：支持 PDF、DOCX、PPTX、XLSX、TXT、MD 等多种文件格式导入。
多种数据源支持：支持本地文件、网址、站点地图甚至手动输入内容作为知识库源。
知识库导出：支持将处理好的知识库导出并分享给他人使用。
支持搜索检查：知识库导入后，用户能实时检索测试，查看处理结果和分段效果。


特色聚焦功能：

快捷问答：在任何场景（如微信、浏览器）中呼出快捷助手，快速获取答案。
快捷翻译：支持快速翻译其他场景中的词汇或文本。
内容总结：对长文本内容进行快速总结，提升信息提取效率。
解释说明：无需复杂提示词，一键解释说明不懂的问题。


数据保障：

多种备份方案：支持本地备份、WebDAV 备份和定时备份，确保数据安全。
数据安全：支持全本地场景使用，结合本地大模型，避免数据泄漏风险。




如何使用Cherry Studio

下载客户端：访问 Cherry Studio 官网（https://www.cherry-ai.com/）。根据操作系统（Windows、macOS、Linux）选择合适的客户端版本进行下载。
安装客户端：

Windows：下载安装包后，双击运行安装程序，按照提示完成安装。
macOS：

下载安装包后，双击打开 .dmg 文件。
将 Cherry Studio 拖动到“应用程序”文件夹中。
打开“应用程序”文件夹，启动 Cherry Studio。


Linux：下载 .deb 或 .rpm 安装包，根据 Linux 发行版选择合适的安装包类型，通过终端运行安装命令。


配置 API 密钥：启动 Cherry Studio 后，配置 API 密钥使用功能。在 Cherry Studio 的设置中，输入 API 密钥。
使用核心功能：

多模型对话：在对话界面中输入问题，选择不同的模型进行回答。
知识库管理：通过导入文件或网址创建本地知识库，使用搜索功能快速查找内容。
AI 绘画：在绘画面板中输入描述，生成高质量图像。
翻译功能：在翻译面板中输入文本，进行翻译。


更多功能：

GitHub：访问 Cherry Studio GitHub 仓库（https://github.com/CherryHQ/cherry-studio）获取最新代码和参与贡献。
GitCode：访问 Cherry Studio GitCode 仓库（https://gitcode.com/CherryHQ/cherry-studio）获取国内镜像和社区支持。



Cherry Studio的应用场景

知识管理与学习：学生和研究人员快速整理和检索文献资料，提升学习和研究效率。
创意设计与绘画：设计师和艺术家用自然语言描述生成高质量图像，激发创意，快速完成设计概念图和艺术作品。
跨语言交流与办公：用户在国际交流、阅读外文文献或处理多语言文档时快速获取准确翻译，提升办公效率。
内容创作与写作：内容创作者快速生成文案、故事、代码片段等，对比不同模型的输出，选择最适合的内容。
个人效率提升：在日常生活中，用户快速获取信息、翻译文本、总结长文，提升个人效率和生活质量。
