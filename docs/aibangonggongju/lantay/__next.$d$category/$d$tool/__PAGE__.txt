                Lantay是什么
Lantay 是面壁智能推出的专业级文档处理智能工作台，被誉为”文档领域的Cursor”。Lantay首创Vibedocing交互范式，采用导航区、工作区、对话区三合一界面，支持单次批量处理200个文档（涵盖PDF、Word等16+格式）。Lantay内置法律、金融等垂直领域专业Agent和可追溯知识库，有效减少AI幻觉。Lantay 是专为律师、分析师等高严谨、高负荷文档工作者打造的人机协作新范式。

Lantay的主要功能

智能工作空间：本地文件夹可一键转化为智能工作空间，实现本地与云端的双向实时同步，内置浏览器支持直接查资料存入工作区。
全量文档处理：支持PDF、Word、PPT、录音等16种以上格式的批量处理，单次最多可处理200个文件（单文件上限100MB，音频可达500MB），实现跨文件的信息提取、汇总与重编。
交互与协作：采用导航区、工作区、对话区三合一的Vibedocing交互界面，支持通过@符号精准关联指定文件分析，提供语音输入和智能录音转写会议纪要功能。
专业Agent生态：首创线性Agent画布，通过输入#可调取法律、金融等领域的专业预置Agent，同时支持用户自定义专属SubAgent并接入专业知识库以减少AI幻觉。
安全与可控：基于端云协同架构提供一体机私有化部署方案，确保敏感数据不上传云端，并通过限制文件来源范围与删改策略保障重要文档不被误操作。

如何使用Lantay

获取与安装：访问官网 https://lantay.modelbest.cn/ ，下载对应版本（支持macOS Apple/Intel芯片及Windows），安装后注册登录。
创建工作空间：在导航区点击连接本地文件夹，一键将电脑中的案件资料或项目文件夹转为智能工作空间，本地与云端实时双向同步。
处理文档


单篇处理：直接上传或拖拽文件至工作区，选中文字可让AI分析、润色或核查风险。


批量处理：同时上传最多200个文件（PDF、Word、录音等16种格式），AI自动跨文件整合汇总。


精准关联：在对话区输入 @文件名 让AI仅基于该文档回答，避免幻觉。



调用专业Agent：在对话框输入 # 唤醒预置Agent（如法律助手、尽调助手），或输入 #自定义Agent名称 调用专属工作流，AI按预设步骤自动执行复杂任务。
语音交互：点击语音按钮直接口述指令，如”结合这三份录音生成合作协议”，系统自动转写录音并生成文档。

Lantay的关键信息和使用要求


开发商：由面壁智能（ModelBest）研发，是国内领先的大模型技术公司。


产品定位：定位为”类 Cursor 的专业级文档智能体工作台”，首创 Vibedocing（Vibe + Docing）交互范式。


核心能力：支持单次批量处理 200 个文件（涵盖 PDF、Word、PPT、录音等 16 种以上格式），具备跨文件智能整合、首创的线性 Agent 画布（输入 # 调用专业 Agent），以及端云协同架构。


目标用户：主要面向律师、金融分析师、专业作者、投资负责人等高严谨、高负荷、对数据安全要求极高的文档密集型行业从业者。


部署方式：目前提供 SaaS 云端版本（公测中）和一体机私有化部署两种方案，满足不同安全级别需求。


系统要求：客户端支持 macOS（区分 Apple 芯片与 Intel 芯片版本）和 Windows 操作系统。


Lantay的核心优势


专业级定位优势：作为”类 Cursor 的文档智能体工作台”，首创 Vibedocing 交互范式，将 AI 能力深度嵌入法律、金融等高严谨行业的工作流，而非仅提供通用问答。


超大规模处理能力：单次可批量处理 200 个文件（涵盖 16 种以上格式），支持跨文档智能整合与溯源分析，远超常规 AI 助手的单文件处理限制。


深度行业适配：内置法律、金融等领域专业 Agent 与可追溯知识库，检索过程全程可见，有效消除通用大模型的”幻觉”问题，确保专业输出有据可依。


极致数据安全：采用端云协同架构，支持一体机私有化部署，敏感数据可完全本地化处理不上云，满足律所、投行等机构的合规要求。


高效交互体验：导航区、工作区、对话区三合一界面设计，配合线性 Agent 画布（# 调用）与 @文件 精准关联，实现无需切换窗口的沉浸式人机协作。


灵活扩展能力：支持用户自定义 SubAgent（类 Skills）和专属工作流，能调用预置专业智能体，可根据个性化需求配置自动化任务流程。

Lantay的应用场景


法律实务场景：律师可用于案件材料批量整理（单次处理 200 份证据）、合同风险审查与条款比对、庭审大纲自动生成、嫌疑人笔录分析，以及结合法规库和类案库进行诉讼策略研究。


金融投资场景：投资负责人和分析师可进行海量尽调数据的本地化处理（支持 500MB 大音频访谈转写）、多期财报数据跨文件整合、VC/PE 尽职调查报告自动撰写，确保敏感商业数据不上云。


企业合规与咨询：适用招投标文件批量解析、多部门项目文档集中管理、合规风险审查，以及结合内部知识库生成标准化报告，支持本地文件夹实时同步与版本控制。


学术研究与出版：专业作者和研究人员可进行文献综述撰写（基于已有文献内容）、学术论文逻辑梳理与润色、多源资料交叉验证，以及书稿批量审校，有效减少 AI 幻觉对专业内容的干扰。
