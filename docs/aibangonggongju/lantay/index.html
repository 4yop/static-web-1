Lantay

AI办公工具

面壁智能推出的文档处理智能工作台

Lantay是什么 Lantay 是面壁智能推出的专业级文档处理智能工作台，被誉为”文档领域的Cursor”。Lantay首创Vibedocing交互范式，采用导航区、工作区、对话区三合一界面，支持单次批量处理200个文档（涵盖PDF、Word等16+格式）。Lantay内置法律、金融等垂直领域专业Agent和可追溯知识库，有效减少AI幻觉。Lantay 是专为律师、分析师等高严谨、高负荷文档工作者打造的人机协作新范式。 Lantay的主要功能 智能工作空间：本地文件夹可一键转化为智能工作空间，实现本地与云端的双向实时同步，内置浏览器支持直接查资料存入工作区。 全量文档处理：支持PDF、Word、PPT、录音等16种以上格式的批量处理，单次最多可处理200个文件（单文件上限100MB，音频可达500MB），实现跨文件的信息提取、汇总与重编。 交互与协作：采用导航区、工作区、对话区三合一的Vibedocing交互界面，支持通过@符号精准关联指定文件分析，提供语音输入和智能录音转写会议纪要功能。 专业Agent生态：首创线性Agent画布，通过输入#可调取法律、金融等领域的专业预置Agent，同时支持用户自定义专属SubAgent并接入专业知识库以减少AI幻觉。 安全与可控：基于端云协同架构提供一体机私有化部署方案，确保敏感数据不上传云端，并通过限制文件来源范围与删改策略保障重要文档不被误操作。 如何使用Lantay 获取与安装：访问官网 https://lantay.modelbest.cn/ ，下载对应版本（支持macOS Apple/Intel芯片及Windows），安装后注册登录。 创建工作空间：在导航区点击连接本地文件夹，一键将电脑中的案件资料或项目文件夹转为智能工作空间，本地与云端实时双向同步。 处理文档 单篇处理：直接上传或拖拽文件至工作区，选中文字可让AI分析、润色或核查风险。 批量处理：同时上传最多200个文件（PDF、Word、录音等16种格式），AI自动跨文件整合汇总。 精准关联：在对话区输入 @文件名 让AI仅基于该文档回答，避免幻觉。 调用专业Agent：在对话框输入 # 唤醒预置Agent（如法律助手、尽调助手），或输入 #自定义Agent名称 调用专属工作流，AI按预设步骤自动执行复杂任务。 语音交互：点击语音按钮直接口述指令，如”结合这三份录音生成合作协议”，系统自动转写录音并生成文档。 Lantay的关键信息和使用要求 开发商：由面壁智能（ModelBest）研发，是国内领先的大模型技术公司。 产品定位：定位为”类 Cursor 的专业级文档智能体工作台”，首创 Vibedocing（Vibe + Docing）交互范式。 核心能力：支持单次批量处理 200 个文件（涵盖 PDF、Word、PPT、录音等 16 种以上格式），具备跨文件智能整合、首创的线性 Agent 画布（输入 #
