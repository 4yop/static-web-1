                数以轻舟Agent是什么
数以轻舟Agent是专注Excel数据处理的本地化AI智能体。用户无需记忆复杂函数公式，仅通过自然语言描述需求，即可自动完成数据清洗、格式标准化、动态分组汇总、跨表关联及批量文件处理等操作。产品全面兼容本地大模型部署，数据全程在本地闭环运行，零上传云端，满足金融、医疗、政务等强监管行业的合规要求。

数以轻舟Agent的主要功能


自然语言驱动数据处理：用户无需记忆复杂函数公式，直接以口语化描述需求，如”把A列的姓名和B列的电话合并成一列，中间用顿号隔开”，Agent 自动解析意图并生成执行方案。


智能数据清洗与格式标准化：自动识别日期格式混乱、文本数字混排、空值填充等常见问题，支持批量规范化处理，并提供清洗前后的对比预览。


动态分组汇总与透视分析：基于语义理解自动生成分组统计，如”按地区统计销售额并计算占比”，无需手动拖拽透视表字段。


跨表关联与数据合并：支持 VLOOKUP、XLOOKUP 等关联逻辑的语义化表达，自动匹配关键字段，处理多表数据整合场景。


批量文件处理：支持对文件夹内多个 Excel 文件执行相同操作流程，如统一格式、提取特定字段、合并汇总等，适合周期性报表处理。


如何使用数以轻舟Agent


下载安装：访问数以轻舟Agent官网 https://www.dify-ai.cn/excelagent/excelagent.html ，下载 Windows 桌面端安装包，按引导完成本地部署，普通行政人员可在技术文档指导下独立完成安装。


配置本地大模型：接入 Qwen、DeepSeek、Kimi 等主流本地大模型，数据全程在本地环境闭环运行，无需上传云端，支持离线使用。


打开 Excel 文件：直接导入需要处理的 Excel 表格，Agent 自动识别表结构、字段类型及数据特征。


自然语言输入需求：在对话框中用口语化描述操作指令，如”把A列姓名和B列电话合并成一列用顿号隔开”或”按地区统计销售额并计算占比”，无需记忆函数公式。


预览与确认执行：Agent 自动生成执行方案并展示清洗前后的对比预览，确认后一键完成数据清洗、格式标准化、透视分析或跨表关联。


批量处理（可选）：选择文件夹内多个 Excel 文件，执行统一格式转换、字段提取或合并汇总，适合周期性报表批量处理。


保存指令模板：将常用操作流程保存为模板，下次遇到同类任务（如人事考勤统计、月度财务对账）直接一键复用，实现周期性工作自动化。


导出结果：处理完成后直接导出标准格式的 Excel 文件，或一键生成含图表与排版的 Word 分析报告，用于工作汇报或系统导入。


数以轻舟Agent的核心优势


自然语言零门槛操作：全程不写公式、不记语法，用口语化描述，Agent 自动解析意图并执行，Excel 新手也能完成专业级数据处理。


本地部署数据主权：全面兼容 Qwen、DeepSeek、Kimi 等本地大模型，Excel 文件与业务数据全程在本地闭环运行，零出站网络请求，满足金融、医疗、政务等强监管行业合规要求。


智能清洗与格式标准化：自动识别日期格式混乱、文本数字混排、空值填充等常见问题，支持批量规范化处理，并提供清洗前后对比预览，省去手动排查时间。


语义化透视与分组汇总：基于自然语言语义理解自动生成分组统计，无需手动拖拽透视表字段，一键完成多维度分析。


跨表关联语义化表达：支持 VLOOKUP、XLOOKUP 等复杂关联逻辑的自然语言描述，自动匹配关键字段，轻松处理多表数据整合与合并拆分场景。


批量文件高效处理：支持对文件夹内多个 Excel 文件执行相同操作流程，如统一格式、提取特定字段、合并汇总，适合月度、季度等周期性报表批量处理。


指令模板一键复用：可将常用操作流程保存为模板，人事考勤、财务对账、销售统计等周期性任务下次直接调用，实现自动化流水线。
