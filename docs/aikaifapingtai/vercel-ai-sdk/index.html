Vercel AI SDK

AIејҖеҸ‘е№іеҸ°

VercelејҖжәҗзҡ„жҗӯе»әAIиҒҠеӨ©жңәеҷЁдәәзҡ„ејҖеҸ‘еҘ—д»¶пјҢж”ҜжҢҒReact/Svelte/VueзӯүжЎҶжһ¶

Vercel AI SDKжҳҜеүҚз«ҜзҪ‘з«ҷејҖеҸ‘е’Ңжүҳз®Ўе№іеҸ°еҸҠNext.jsејҖеҸ‘еӣўйҳҹгҖҢVercelгҖҚжҺЁеҮәзҡ„пјҢз”ЁдәҺеҝ«йҖҹжһ„е»әAIиҒҠеӨ©жңәеҷЁдәәзҪ‘з«ҷеә”з”ЁзЁӢеәҸзҡ„ејҖеҸ‘еҘ—д»¶пјҢеҸҜд»Ҙеё®еҠ©ејҖеҸ‘дәәе‘ҳдҪҝз”ЁJavaScriptе’ҢTypeScriptжһ„е»әеҜ№иҜқејҸзҡ„AIз”ЁжҲ·з•ҢйқўгҖӮ Vercel AI SDKзҡ„зү№жҖ§ ж”ҜжҢҒReact/Next.jsгҖҒSvelte/SvelteKitе’ҢVue/NuxtзӯүеүҚз«ҜжЎҶжһ¶пјҢд»ҘеҸҠNode.jsгҖҒServerlessе’ҢEdge Runtime еҶ…зҪ®еҗ„з§ҚAIжЁЎеһӢзҡ„йҖӮй…ҚеҷЁпјҢж”ҜжҢҒLangChainгҖҒOpenAIгҖҒAnthropicе’ҢHugging FaceзӯүжҸҗдҫӣзҡ„еӨ§иҜӯиЁҖжЁЎеһӢ жҸҗдҫӣдәӨдә’ејҸеңЁзәҝжҸҗзӨәplaygroundпјҲsdk.vercel.aiпјүпјҢе…¶дёӯеҢ…еҗ«20дёӘејҖжәҗе’Ңдә‘LLMгҖӮеҸҜд»Ҙе®һж—¶еұ•зӨәдёҚеҗҢеҜ№иҜқжЁЎеһӢзҡ„иҒҠеӨ©з•ҢйқўпјҢе№¶дё”еҸҜд»Ҙеҝ«йҖҹз”ҹжҲҗд»Јз ҒгҖӮ жҸҗдҫӣеӨҡдёӘAIиҒҠеӨ©жңәеҷЁдәәзҡ„жЁЎжқҝе’ҢзӨәдҫӢпјҢдҪ еҸҜд»Ҙе…ӢйҡҶ/еӨҚеҲ¶VercelжҸҗдҫӣзҡ„еҹәдәҺдёҚеҗҢжЎҶжһ¶е’ҢжЁЎеһӢејҖеҸ‘зҡ„AIиҒҠеӨ©жңәеҷЁдәәзҡ„еҲқе§ӢжЁЎжқҝ еҰӮдҪ•дҪҝз”ЁVercel AI SDK еүҚжҸҗжқЎд»¶йңҖиҰҒеңЁз”өи„‘дёҠе®үиЈ…Node.js 18+зүҲжң¬пјҢеҰӮжһңиҰҒејҖеҸ‘еҹәдәҺOpenAIзҡ„GPTиҒҠеӨ©жңәеҷЁдәәпјҢйңҖиҰҒиҺ·еҫ—OpenAI APIеҜҶй’Ҙ дҪҝз”ЁNext.jsпјҲpnpm dlx create-next-app my-ai-appпјүжҲ–иҖ…SvelteпјҲpnpm create svelte@latest my-ai-appпјүзӯүжЎҶжһ¶еҲӣе»әдёҖдёӘе…Ёж–°зҡ„йЎ№зӣ®пјҢе№¶е®ҡдҪҚеҲ°еҲӣе»әеҘҪзҡ„зӣ®еҪ•пјҲcd my-ai-appпјү е®үиЈ…дҫқиө–йЎ№пјҢpnpm install ai openai-edge й…ҚзҪ® OpenAI API еҜҶй’ҘпјҢ.env.localеңЁйЎ№зӣ®ж №зӣ®еҪ•дёӯеҲӣе»әдёҖдёӘж–Үд»¶е№¶ж·»еҠ жӮЁзҡ„ OpenAI API еҜҶй’Ҙ еҲӣе»әAPIи·Ҝз”ұе№¶иҝһжҺҘUIпјҢе®ҢжҲҗеҗҺдҪҝз”Ёpnpm run devиҝҗиЎҢеҗҜеҠЁеә”з”ЁзЁӢеәҸ
