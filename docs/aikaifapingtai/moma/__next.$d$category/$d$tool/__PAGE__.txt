                MoMA是什么
MoMA是中国移动推出的国内首个开放普惠大模型聚合平台，汇聚超 300 款主流大模型（开源、闭源、自研九天），提供统一 API 网关与智能路由能力。用户无需逐个注册账号，通过一个平台可实现模型体验、API 调用、一键部署与微调，支持按量/一次性计费，提供 2500 万 Token 免费额度，大幅降低从”技术尝鲜”到”规模应用”的门槛。

MoMA的主要功能


模型广场：汇聚 300+ 款 AI 模型，覆盖文本、视觉、语音、向量、多模态等类别，支持在线体验、API 调用、一键部署与微调。


统一 API 网关：提供标准化接口，一次接入即可调用所有模型，无需适配不同厂商协议。


智能路由（九天 inside）：自动分析用户需求，动态匹配最佳模型，支持成本优先、效果优先、均衡优先三种策略，故障时秒级容灾切换。


RAG 检索增强生成：提供多源知识库构建能力，支持混合检索与多路召回，有效整合企业内外部海量碎片化知识资源，提升生成内容准确性。


智能体开发：基于 LLM 自主规划和工作流的智能体编排方式，满足无代码/低代码智能体开发需求，快速构建 AI Agent。


智能体商店：汇聚多领域现成智能体，可直接体验效果；同时支持将自己开发的智能体发布到商店，拓展应用渠道。


可视化工作流编排：通过拖拽方式对插件、模型、代码、知识库等功能进行组合编排，实现复杂稳定的业务流程自动化。


Token 计量与透明计费：提供每笔 Token 消耗明细、时延、吞吐性能监控，支持按量计费与一次性计费。


如何使用MoMA

进入控制台：访问MoMA官网 https://ecloud.10086.cn/portal/product/MaaS ，完成账号登录和注册进入控制台。
浏览模型广场：打开模型广场查看 300+ 款模型，可按类别、功能、系列筛选，或直接搜索目标模型。
体验或调用模型：选择单款模型点击「在线体验」即时测试，或点击API 调用获取接口地址与密钥。
配置智能路由： 进入智能路由模块，选择策略，勾选模型集，创建自动调度规则。
构建 RAG 知识库：进入 RAG 模块，上传企业文档构建多源知识库，配置混合检索策略增强生成效果。
开发智能体： 进入智能体开发模块，基于 LLM 自主规划能力，通过无代码/低代码方式编排工作流，构建专属智能体。
发布到智能体商店：将开发完成的智能体发布到商店，供团队内部或外部用户直接体验和使用。

MoMA的核心优势


模型覆盖最全：国内最大模型服务平台，300+ 款模型覆盖开源、闭源、自研九天及机密模型。


智能降本：智能路由自动选择性价比最优模型，单位 Token 成本降低约 30%，资源占用降低 50% 以上。


全栈 AI 能力：提供模型调用，集成 RAG 知识增强、智能体开发、工作流编排，覆盖从底层模型到上层应用的全链条。


低代码智能体开发：无需深厚编程基础，通过可视化编排即可构建和发布智能体，降低 AI 应用开发门槛。


高可用保障：秒级自动容灾机制，单点故障不影响业务，适合生产环境。


计费透明：按实际 Token 消耗付费，提供明细级监控，告别”按包计费”的糊涂账。


免费试用：提供 2500 万 Token 免费额度，支持先试用后付费。

企业 AI 中台：作为统一模型入口，支撑内部多个业务线的智能化需求，避免重复对接。
