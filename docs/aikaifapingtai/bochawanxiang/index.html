博查万象

AI开发平台

博查多模态混合搜索和语义排序API开放平台

博查万象是什么 博查万象是基于多模态混合搜索和语义排序技术的新一代搜索引擎，提供Web Search、AI Search、Semantic Reranker等API服务。博查万象基于多模态混合搜索与语义排序技术，从近百亿网页中检索新闻、图片、视频、百科等高质量内容，支持自然语言查询与0.15秒极速响应，数据不出海且符合国内安全规范，助力AI Agents、Chatbots及RAG应用降低幻觉、连接实时世界知识。 博查万象的主要功能 Web Search API（通搜）：支持自然语言搜索全网，返回标题、URL、摘要、网站图标、发布时间、图片等结构化数据，单次最多50条，响应时延低至0.15秒。 AI Search API（AI搜）：混合搜索全网并自动获取垂域结构化数据，支持大模型实时生成答案、追问问题与流式输出，返回多模态参考源与总结答案。 Semantic Reranker API（语义排序）：自研80M参数Transformer模型，在BM25/RRF初排后对候选结果语义精排，支持recall50@top_n50排序，结果更契合LLM偏好。 Agent Search API（智能体搜索）：调用指定AI智能体回答用户问题，返回专业深度答案与追问问题（已结束内测，垂直领域需求可转用AI Search）。 Bocha Hybrid Search Database（BochaDB）：支持关键字+向量混合搜索，内置语义重排序能力，为RAG应用提供一站式检索基础设施。 余额查询API：实时查询账户余额与资源消耗情况。 如何使用博查万象 注册并获取 API-KEY：访问博查万象官网 https://open.bochaai.com/ 注册账号，免费领取调用资源包并获取 API-KEY，用于后续接口认证。 选择接入方式：平台支持四种调用方式：官方 REST API、大模型 Function Call（兼容 OpenAI / LangChain）、AI 开发平台插件（腾讯元器、扣子、钉钉、Dify）以及 MCP Server。 调用 Web Search API（通搜）：通过 https://api.bocha.cn/v1/web-search 进行全网搜索，支持自然语言查询、指定时间范围、限定网站范围与输出文本摘要。 调用 AI Search API（AI 搜）：进行多模态混合搜索，自动获取垂域结构化数据，支持大模型实时生成答案、追问问题与流式输出，返回网页、图片、多模态参考源（模态卡）及总结答案。 调用 Semantic Reranker API（语义排序）：对已有候选文档使用自研 bocha-semantic-reranker 进行语义精排，支持 recall50@top_n50 排序，输出相关性得分与排序结果，使结果更契合 LLM 偏好。 查询账户余额：通过余额查询 API 实时查看账户资源消耗与剩余额度，便于成本管理与用量监控。 博查万象的关键信息和使用要求 API端点：https://api.bocha.cn/v1/web-search 等，RESTful JSON格式。 认证方式：Bearer Token（API-KEY）。 数据覆盖：近百亿网页与生态内容源，涵盖新闻、图片、视频、百科、机酒、学术等垂类。 合规安全：数据来源与搜索结果符合国内安全规范，数据不出海，满足B端企业合规需求。 响应性能：Web Search支持0.15秒极速响应；AI Search支持流式与非流式两种输出。 返回上限：单次搜索最多返回50条网页结果（count=50）。 接入生态：提供官方API、Function Call、AI开发平台插件、MCP Server四种调用方式。 博查万象的核心优势 专为AI优化：搜索结果格式专为大模型设计，Semantic Reranker与LLM同为Transformer架构，排序后的结果更受LLM青睐。 混合搜索精准：意图识别Router驱动关键字+向量多路召回，经BM25/RRF初排与语义精排，全链路提升搜索准确性。 轻量高效：自研80M参数语义排序模型，性能接近世界一线280M/560M参数模型，推理速度更快、成本更低。 合规领先：国内稀缺的 content-compliant 搜索API，数据不出海，填补国内市场空白。 极速响应：Web Search支持0.15秒响应时延，满足高并发实时检索场景。 生态集成完善：原生支持OpenAI/LangChain Function Call、MCP Server，提供腾讯元器、扣子、钉钉、Dify等智能体平台插件。 博查万象的同类竞品对比 对比维度 博查AI开放平台 Tavily Bing Web Search API 产品定位 给AI用的世界知识搜索引擎 AI专用搜索API
