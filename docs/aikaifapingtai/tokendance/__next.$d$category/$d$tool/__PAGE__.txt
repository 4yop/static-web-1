                TokenDance是什么
TokenDance（词元跳动）是观猹团队推出的一站式大模型 API 调用平台，定位为”统一模型 API 网关”。TokenDance 兼容 OpenAI、Claude、Gemini 等原生协议，通过单一 Base URL 即可接入 MiniMax、通义千问、Kimi、智谱、DeepSeek 等国内主流模型，提供智能路由、统一计费、容错降级等企业级能力，帮助 AI 开发者降低多供应商管理成本，实现分钟级零迁移接入。

TokenDance的主要功能


多协议兼容：原生支持 OpenAI、Claude、Gemini 协议，切换 Base URL 可接入，无需修改现有代码。


智能路由：根据模型名称自动路由至对应供应商端点，单一入口屏蔽底层调度细节。


统一计费：跨供应商统一 Token 消耗统计与账单管理，告别多平台分别充值。


容错降级：同一模型支持多供应商端点自动切换；单次请求可指定多个候选模型逐级降级，保障服务持续可用。


模型丰富：已接入 8+ 国内头部模型，包括 MiniMax、通义千问、Kimi、智谱、DeepSeek 等，持续扩展中。


开箱即用：支持 Watcha 一键登录，兼容现有 SDK，零迁移成本，分钟级完成接入。


如何使用TokenDance


访问官网：访问TokenDance官网 https://tokendance.space/ ，用 Watcha 一键登录完成账号注册与认证。


获取 API Key：在控制台生成个人 API Key。


修改 Base URL：将现有项目中的 Base URL 替换为 https://tokendance.space/gateway/v1/。


选择模型：在请求中填入目标模型 ID（如 DeepSeek、Kimi 等）。


发起调用：使用现有 OpenAI SDK 或 curl 直接发起请求，即可通过统一网关调用多供应商模型。


申请补贴（可选）：若需大量 Token，可填写”百亿补贴计划”申请表，根据产品阶段获取赞助额度。

统一成本管控：提供跨供应商的 Token 消耗统计与统一账单，彻底告别多平台分别充值、对账混乱的成本管理难题。

TokenDance的应用场景


AI 产品快速验证：初创团队通过统一网关低成本试用多个国内大模型，快速找到最适合自身业务的模型组合。


多模型策略部署：生产环境根据任务类型（推理、创作、代码）动态路由至不同供应商的最优模型。


企业级 Agent 开发：为消耗大量 Token 的 Agent 系统提供高可用、可降级的底层模型基础设施。


成本敏感型应用：通过补贴计划与统一计费，帮助教育、科研、个人开发者降低 Token 开销。
