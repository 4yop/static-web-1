观猹团队推出的一站式大模型 API 调用平台

TokenDance是什么 TokenDance（词元跳动）是观猹团队推出的一站式大模型 API 调用平台，定位为”统一模型 API 网关”。TokenDance 兼容 OpenAI、Claude、Gemini 等原生协议，通过单一 Base URL 即可接入 MiniMax、通义千问、Kimi、智谱、DeepSeek 等国内主流模型，提供智能路由、统一计费、容错降级等企业级能力，帮助 AI 开发者降低多供应商管理成本，实现分钟级零迁移接入。 TokenDance的主要功能 多协议兼容：原生支持 OpenAI、Claude、Gemini 协议，切换 Base URL 可接入，无需修改现有代码。 智能路由：根据模型名称自动路由至对应供应商端点，单一入口屏蔽底层调度细节。 统一计费：跨供应商统一 Token 消耗统计与账单管理，告别多平台分别充值。 容错降级：同一模型支持多供应商端点自动切换；单次请求可指定多个候选模型逐级降级，保障服务持续可用。 模型丰富：已接入 8+ 国内头部模型，包括 MiniMax、通义千问、Kimi、智谱、DeepSeek 等，持续扩展中。 开箱即用：支持 Watcha 一键登录，兼容现有 SDK，零迁移成本，分钟级完成接入。 如何使用TokenDance 访问官网：访问TokenDance官网
