                千问云是什么
千问云是阿里云推出的为AI Agent而生的全新MaaS模型服务平台，聚合150余款主流大模型API，支持将模型服务全面封装为Skills与CLI工具，让Agent和开发者通过一句指令可完成模型选型、调用与用量管理，驱动AI生产力进入 Agentic 时代。

千问云的主要功能

模型广场一站式接入：聚合Qwen、GLM、Kimi、DeepSeek、Wan、HappyHorse等150余款主流及第三方模型，覆盖文本、图像、视频、语音全模态。
多维模型对比选型：支持按参数规模、能力维度、价格、上下文长度、模态支持、适用场景等属性横向对比，对比后可直接进入体验页面用真实Prompt验证输出效果。
Skills技能包：将模型选型、认证配置、API调用、用量查询等完整链路封装为Agent可理解的技能组合，Agent可动态路由不同模型，自动调用视觉、生图、视频等能力，用户无需编写代码。
CLI命令行工具：提供登录认证、模型查询、模型调用、用量查询、环境诊断等命令行能力，支持通过脚本自动化完成所有模型服务工作流。
智能用量管理：Agent可实时拉取模型用量数据，自动分析消耗趋势、识别异常并输出成本优化建议，过程数据通过CLI日志透明呈现。
Token Plan订阅：面向高频AI编程与Agent工具场景推出订阅制套餐，一个API Key全模态通行，兼容主流AI工具，月费可控且成本较按量付费显著降低。

如何使用千问云

访问官网：打开千问云官网 https://www.qianwenai.com/ ，浏览模型广场并对比所需模型的参数、能力与价格。
安装CLI工具：在终端执行 npm install -g @qianwenai/qianwen-cli 安装全局命令行工具。
配置Skills：执行 npx skillsadd QianWen-AI/qianwen-ai 为Agent安装千问云技能包，或访问平台提供的Skills.md文档按说明配置。
认证与调用：通过CLI完成登录认证，获取API Key后可用一句指令让Agent自动完成模型选型、调用与任务执行。
用量监控与优化：通过CLI实时查看模型用量、账单与免费额度，Agent将自动分析趋势并提供成本优化建议。

千问云的核心优势

国内首个深度面向Agent的模型服务平台：从”为人服务”转向”为Agent服务”，重构模型平台的核心交互范式。
开源生态：Skills与CLI工具包已在GitHub开源，开发者可自由扩展与集成。
多模型兼容：一个API Key通行150余款模型，避免多平台切换与多密钥管理。
极致性价比：Token Plan订阅制低至6折，百万Tokens输入最低仅需1元，显著降低高频Agent场景成本。
人机协同灵活：支持Agent自主调用，保留人工在线试用、对比选型、Prompt调试的交互入口。

千问云的应用场景

AI编程开发：开发者通过Qoder等编程Agent直接调用千问云CLI，自动完成模型选型、代码生成与部署。
多模态内容创作：依托万相能力，通过Agent自动调度文生图、文生视频、图生视频模型，实现影视级内容批量生产。
企业智能办公：集成悟空等工作助手，一句话调动多个应用完成日程、文档、审批、会议等任务。
智能座舱与出行：依托千问大模型集成出行助手、智能规划、长期记忆等能力，构建安全愉悦的车载交互体验。
社交与虚拟陪伴：面向社交拟人交互场景，支持虚拟IP打造与实时情感化对话，提供沉浸式交互新体验。

千问云的常见问题
Q：同一订阅能否在多个工具中使用？

可以。同一 API Key 可在全部兼容的 AI 工具中使用，额度共享消耗。

Q：Token Plan 是否支持退订？

Token Plan 团队版不支持退订。

Q：Token Plan 团队版的 API Key 能与其他 API Key 混用吗？

不能。Token Plan 团队版、Coding Plan 和按量计费三者的 API Key 和 Base URL 互不相通，请勿混用。误用其他 API Key 不会抵扣 Token Plan 团队版的套餐额度。

Q：Token Plan 有哪些使用限制？

仅限在兼容的 AI 编程和智能体工具中交互式使用，不可用于自动化脚本或应用后端。违规使用可能导致订阅暂停或 API Key 封禁。
