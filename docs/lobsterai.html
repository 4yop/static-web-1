LobsterAI是什么 LobsterAI 是网易推出的全场景办公助手 Agent，也是国内大厂首个开源桌面级 Agent。强调本地优先、安全可信，用户的会话、配置和记忆主要保留在本机，工具调用过程可控可审计。通过连接本地文件、终端、浏览器和项目环境，帮助用户完成文档处理、数据分析、网页生成、PPT 制作、浏览器自动化、定时任务等真实办公任务。 LobsterAI的主要功能 桌面级 Agent 办公助手：可连接文件、终端、浏览器与本地项目，进入真实工作环境完成复杂办公任务。 任务自动拆解与执行：用户用自然语言描述需求后，自动拆解步骤并逐步完成，例如搭建进销存系统、制作数据看板、整理简历筛选表等。 文档与内容创作：支持内容总结、报告撰写、PPT 制作、文案生成、资料整理等场景。 数据处理与网页生成：可读取 Excel、PDF 等本地文件，完成数据分析、可视化页面制作、表格整理和信息提炼。 浏览器自动化：可打开网页后台、查看数据、检查异常并整理结论，适合运营、投放、数据监控等重复性工作。 定时任务：支持设置周期性任务，例如每天整理 AI 科技新闻、定时检查广告后台数据等。 多 Agent 与子 Agent：支持创建股票助手、内容创作、备课出卷等专用 Agent，并通过子 Agent 协同处理复杂任务。 专家套件：按工作场景安装能力包，将相关技能、MCP 服务或连接器组合起来，在对话中直接启用；例如内置电脑操作套件可用于本地桌面应用控制。 技能系统：提供细粒度工作流能力，并支持自定义技能扩展具体任务。 MCP 支持：通过 MCP 服务接入外部工具与数据源，扩展 Agent 可调用的工具边界。 远程指挥电脑干活：支持通过微信、企微、钉钉、飞书等 IM 工具，远程指挥 LobsterAI 在电脑上执行任务。 如何使用LobsterAI 下载安装：访问LobsterAI 官网 https://lobsterai.youdao.com/ ，下载客户端，提供 MacOS Apple 芯片、MacOS Intel 芯片和 Windows 版本。 直接对话：安装后打开客户端，在输入框中直接描述任务或提出问题，例如”帮我整理这个文件夹里的简历””根据这个 Excel 做一个可视化页面””每天早上 9 点整理 AI 新闻”。 权限授权：根据任务需要授权 LobsterAI 访问本地文件、浏览器、终端或相关工具；执行过程中可查看工具调用记录，确保操作可控。 进阶配置：可使用定时任务、专家套件、技能、MCP、我的 Agent 等功能，创建更适合自己工作流的自动化助手。 LobsterAI的适用人群 办公人群：适合经常处理文档、表格、PPT、资料总结和跨工具协作的用户。 运营与市场人员：适合广告后台检查、数据整理、异常分析、竞品调研、日报周报等工作。 内容创作者：适合选题调研、文案撰写、内容总结、脚本整理和素材规划。 开发者与技术人员：适合本地项目处理、网页生成、自动化脚本、终端操作和开源 Agent 能力探索。 中小企业与个体经营者：适合搭建轻量系统、整理进销存数据、制作本地工具，提升重复性工作的自动化程度。 LobsterAI的产品价格 限时新用户送 100 万 token。 完整套餐与付费价格建议以官网“定价”页面或客户端内实际展示为准。
